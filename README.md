# Michelin Restaurants

A Vite + React + Tailwind CSS app for exploring Michelin-starred restaurants from Wikidata on a searchable list and map.

Live site: <https://louispaulet.github.io/michelin-restaurants/>

## Data

The dataset is generated by `scripts/generate_restaurants_csv.py` into `public/data/restaurants.csv`.

Use `make refresh` to create the CSV only when it does not already exist. Use `make data` to force a fresh download.

The generator queries Wikidata for restaurants that:

- have `award received` (`P166`) = `Michelin star` (`Q20824563`)
- have a `Michelin Restaurants ID` (`P4160`)

It then tries to enrich current 1/2/3-star tiers from Michelin Guide pages using the Michelin ID.

Current limits:

- Restaurants must already have both `award received` = `Michelin star` and a `Michelin Restaurants ID` in Wikidata.
- Restaurants listed by Michelin but missing from Wikidata, or missing `P4160`, are not discovered.
- Star tiers are best effort because Michelin pages are parsed from public HTML.
- The CSV is static until `make data` is run manually or `make refresh` creates a missing file.

## Commands

```bash
make up
make kill
make refresh
make test
make build
make deploy
```

## Development

```bash
npm install
npm run data
npm run dev
```

The app uses `HashRouter`, so deep links such as `#/restaurants` and `#/map` work on GitHub Pages.

## Deployment

`make deploy` runs the production build and publishes `dist` with the `gh-pages` package.
